# 万维钢 · 著作与系统思考

## 一、出版著作

### 《万万没想到：用理工科思维理解世界》（2014）
- **出版社**：电子工业出版社
- **核心论点**：用数据、实验和概率替代直觉和"常识"来理解世界
- **关键概念**：反常识思维、刻意练习、学习区理论、理工科思维 vs 文科思维
- **结构**：反常识思维 → 成功学解药 → 霍金的答案
- **豆瓣评分**：高口碑畅销书

### 《智识分子：做个复杂的现代人》（2016）
- **核心论点**：面对复杂现代世界，需具备跨学科的"智识"能力
- **关键概念**：智识分子（智慧+见识）、狐狸 vs 刺猬思维、三种教育观
- **核心区分**：普通人随大流 → 有意识者按判断选择 → 高手按规律行事

### 《高手：精英的见识和我们的时代》（2017）
- **核心论点**：精英阶层有不同的认知和见识
- **关键概念**：画布策略（学徒为高手创造条件）、系统思维、运气概率管理
- **来源**：精英日课精华集结

### 《你有你的计划，世界另有计划》（2019）
- **核心论点**：用理工科思维揭示看似反常实则靠谱的真实世界
- **内容**：八个不同主题，30万+用户验证
- **豆瓣**：book.douban.com/subject/30489754/

### 《学习究竟是什么》（2020）
- **核心论点**：从认知科学角度系统解析"学习"
- **关键概念**：刻意练习精髓、学习区练习、强力研读法、掌握多领域知识
- **核心引用**："练习的精髓是持续做自己做不好的事"

### 《相对论究竟是什么》（2020）
- **出版社**：新星出版社 / 得到图书
- **豆瓣评分**：8.6
- **定位**：通才丛书系列，用简明方式帮助理解相对论

### 《博弈论究竟是什么》
- **定位**：通才丛书系列

### 《和这个世界讲讲道理：智识分子2020s》（2021）
- **豆瓣评分**：8.2
- **核心**：从物理、数学、哲学、计算机、经济学、大数据、AI等多维度阐述
- **结构**：社会、历史、教育、未来四篇

### 《佛畏系统》（2022）
- **核心论点**："凡夫畏果，菩萨畏因，佛畏系统"
- **关键概念**：系统思维全面提升决策力，不纠结单次结果而建系统

### 《科学思考者》
- **核心论点**：科学思考的本质是从被情绪和惯性推着走到用理性和证据做主
- **三种思维错误**：奇迹思维、愿望思维、故事思维
- **方法论**：批判性思维、奥卡姆剃刀、贝叶斯方法

### 《量子力学》（台湾远流出版）
- **核心**：从物理学角度探讨世界底层逻辑
- **观点**："正因为量子力学中有许多令物理学家不自在的事实，才让世界如此多姿多彩"

### 《拐点：站在AI颠覆世界的前夜》（2024年5月）
- **核心论点**：AI变革是启蒙运动级别的大事件
- **关键概念**：AI的"涌现"能力、GPT-4已是某种AGI、人要比AI凶
- **引用**："2024年我出版了《拐点》一书，认为人类正处在AI颠覆世界的拐点前夜——现在我可以非常负责任地说，我们已经身处拐点之中。"

### 《人比AI凶》（2025）
- **豆瓣评分**：7.8
- **核心论点**："当AI无所不能，你靠什么不可替代？"
- **7大主题**：AI前沿洞察 → AI实操 → 成长之道 → 未来趋势
- **关键观点**：智商越来越不重要，"微决策"才是关键

## 二、精英日课系列（得到App）

- **形式**：音频+图文，日更，每季300+期，连续6季，10万+学员
- **定位**：追踪西方精英在经济学、社会学、心理学、管理学、哲学、科技、商业等领域的思想新突破
- **方法论**：消化理解后用中国读者习惯的方式分享
- **推荐书单**：套装25册（湛庐文化出版），包括《成功与运气》《思维简史》等
- **第六季**：2025年8月25日完结，涵盖AI提示工程、光刻机、五种财富理论、神经科学等

## 三、反复出现的核心论点（≥3次 = 真信念）

1. **理工科思维优于文科思维**：贯穿所有著作
2. **刻意练习 / 学习区理论**：出现于《万万没想到》《学习究竟是什么》《高手》等
3. **狐狸思维优于刺猬思维**：出现于《智识分子》《高手》《和这个世界讲讲道理》
4. **系统思维 > 纠结单次结果**：出现于《佛畏系统》《高手》《你有你的计划》
5. **反常识 / 反鸡汤**：贯穿所有著作的核心姿态
6. **跨学科广博 > 单一领域深耕**：贯穿所有著作
7. **AI时代人的独特价值在于主动性和微决策**：《拐点》《人比AI凶》

## 四、自创术语/概念

- **智识分子**：智慧+见识的现代人，能欣赏复杂、区分理论与实际
- **强力研读**：不好玩、用得上、高强度的读书方法
- **理工科思维**：用数据、实验、概率理解世界的方式
- **佛畏系统**：从系统设计层面规避问题的最高境界
