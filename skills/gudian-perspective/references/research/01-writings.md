# 古典 · 著作与系统思考

## 一、三部曲（层层递进的完整成长体系）

### 1. 《拆掉思维里的墙》（第一部：认知心法）
- **核心论点**：自我全责——如果你全然为自己负责，你就能改变一切
- **七种心智模式**：
  1. 安全感之墙：过早买房束缚职业发展（巴菲特投资自己而非房产）
  2. 成功学之墙：坚持不一定成功，盲目比较是陷阱
  3. 心智模式之墙："我们亲手搬砖、垒砌，把自己围在里边——勤奋却低效"
  4. 兴趣与投入：以兴趣驱动人生规划，"投入当下，积极尝试"
  5. 掌控者模式：从受害者模式转向掌控者模式，做自己人生的"全责"承担者
  6. 比较之墙：只有有钱才能幸福的思维定式
  7. 等待之墙：不要在等待中枯萎，要尝试、要开始
- **来源可信度**：一手（本人著作）

### 2. 《你的生命有什么可能》（第二部：生涯方法论）
- **核心论点**：在热爱的领域努力地玩
- **生涯三叶草模型**：兴趣 → 能力 → 价值 的循环飞轮
  - 缺兴趣 → 厌倦；缺能力 → 焦虑；缺价值 → 失落
- **生涯四度**：高度（成就）、深度（思想）、宽度（经历）、温度（幸福感）
- **核心公式**：自由 = 能力 − 欲望；好生活 = 愉悦 + 有趣
- **来源可信度**：一手（本人著作）

### 3. 《跃迁：成为高手的技术》（第三部：系统论/战略）
- **核心论点**：如何利用规律和趋势放大个人努力（取势、明道、优术）
- **四大板块**：
  1. 高手的暗箱：阶段性非线性成长，调用知识而非记忆，人机合一
  2. 选择战略：专注头部，高价值+高优势领域，二八法则
  3. 联机学习：知识源头→功利读书→知识晶体→知识IPO（Input→Problem→Output）
  4. 破局思维：系统角度思考，找到关键节点
- **来源可信度**：一手（本人著作）

## 二、《超级个体》课程（得到App）

- **目标**：以职业生涯规划的系统方法，让你终身保持职业竞争力
- **核心定义**：能持续自我更新和迭代，不断生产信息、发散力量，不依附于某份工作，靠自身的热爱和能力就能赚钱、发展的人
- **生涯四看模型**：向深发展、向上发展、向外发展、向左右发展
- **来源可信度**：一手（本人课程）
