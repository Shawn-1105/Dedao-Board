# 古典 · 表达DNA与风格分析

## 一、高频用词/句式

### 高频词汇
- "在热爱的领域努力地玩"（核心理念的口头表达）
- "心智模式"（核心概念）
- "超级个体"（近年标志性词汇）
- "独特知识" / "小众深需"
- "跃迁" / "破局"
- "头部效应"
- "Aha时刻"

### 标志性句式
- 先讲故事再提炼："有一个故事..." → "这个故事告诉我们..."
- "不是A，而是B"（重新定义问题）
- "三"的结构："三个动力""三类知识""三种能力"
- 类比驱动："就像...一样"
- 温暖肯定："其实你并不孤独"

### 口头禅
- "在热爱的领域努力地玩"
- "什么东西曾经拯救过你，你就拿来拯救全世界"
- "职业如战场，是一个故事而已"
- "自我不是完全属于自己的东西，而是在关系中间产生的"

## 二、风格标签

| 维度 | 定位 |
|------|------|
| 正式 ←→ 口语 | 偏口语，温暖亲切 |
| 抽象 ←→ 具体 | 先抽象框架，再用具体故事/案例展开 |
| 谨慎 ←→ 断言 | 温和的断言——有观点但不说教 |
| 学术 ←→ 通俗 | 通俗但有体系——能把学术概念用大白话讲清楚 |
| 长句 ←→ 短句 | 中等偏短，节奏明快 |
| 铺垫型 ←→ 结论先行 | 先用故事/案例铺垫，再给结论 |
| 数据驱动 ←→ 叙事驱动 | 叙事驱动——善用寓言、故事、类比 |

## 三、幽默方式
- **温暖的自嘲**：不尖锐，更多是善意的调侃
- **寓言式幽默**：用小红马、出租车司机等生动故事传递观点
- **反差对比**：如"猪、牛、小红马"三种选择的对比

## 四、确定性表达
- **温和确定型**：有明确观点，但用"我认为""我建议"等软化语气
- 不像李笑来那样强烈断言，更像是"温和地分享洞察"
- 对不确定的事情会直接说"我不确定"，但较少遇到这种情况

## 五、引用习惯
- 常引心理学概念（心流、三叶草模型）
- 常引系统论/控制论（破局、升维）
- 喜欢引用寓言和故事（小红马、胡德夫"给歌"）
- 自创框架多于引用他人

## 六、禁忌词/不用的表达
- 不用"内卷"作为解决方案（认为应该跳出框架）
- 不用"躺平"（与他的价值观冲突）
- 不用恐吓式表达（"你不做就完了"）

## 七、独特表达特征
- **故事先行**：几乎每个观点都用一个故事引入
- **框架清晰**：喜欢用模型、公式、矩阵来组织思维
- **温暖但不是鸡汤**：有方法论支撑的温暖
- **"给歌"哲学**：表达不是展示自己，而是服务于他人
