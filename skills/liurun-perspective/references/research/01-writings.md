## 一、主要著作与核心论点

### 1. 《底层逻辑》（2021）/ 《底层逻辑2》（2022）

**核心论点**：
- 世间万事背后都有一个"底层逻辑"，理解底层逻辑比记住具体方法更重要
- 底层逻辑 = 环境变量 + 公理体系 → 推导出结论
- 强调"是非对错的底层逻辑"：事实、观点、立场、信仰四层区分
- "正常成年人只看利弊，小孩子才分对错"——利益驱动的行为分析框架
- 商业底层逻辑：价值创造 > 价值传递 > 价值分配

**反复出现的概念**：
- **系统思考**：要素与连接关系决定系统行为，不只是看要素本身
- **复利思维**：不是线性增长，而是指数增长；关键是"持续"和"足够长的时间"
- **概率思维**：用概率看世界，做大概率对的事情
- **飞轮效应**：正向循环的自我增强系统

**自创/推广术语**：
- "底层逻辑"本身是他最广为人知的标签
- "是非对错的底层逻辑"——事实vs观点的区分法
- "商业模式的本质"系列分析框架

### 2. 《进化的力量》（2022年度演讲主题延伸）

**核心论点**：
- 商业世界遵循进化论：变异→选择→隔离→适应
- 企业要在变化中"进化"而非"优化"
- 适应环境变化的能力比当前状态更重要
- "不是最强壮的物种能存活，也不是最聪明的，而是最能适应变化的"
- 每年通过年度演讲梳理当年的商业进化趋势

**关键框架**：
- 进化四步法：变异（尝试新事物）→ 选择（市场筛选）→ 隔离（独立发展）→ 适应（找到生态位）
- 用生物学视角解释商业现象

### 3. 《5分钟商学院》（2016-2017，分商业篇、管理篇、个人篇、工具篇）

**核心论点**：
- 把商学院知识压缩到5分钟可消化的单元
- 每个概念 = 一个商业原理 + 一个实际案例 + 一个行动建议
- 强调"用得到"而非"知道"——知识要能指导行动

**反复出现的概念**：
- 边际成本、机会成本、沉没成本（经济学三成本）
- 信息不对称、委托代理问题（制度经济学）
- PDCA循环、OKR（管理工具）
- 比较优势、规模效应（战略思维）

### 4. 《每个人的商学院》（系列）

**核心论点**：
- 商业思维不是商学院的专利，每个人都需要
- 用通俗语言解释商业概念
- 强调思维框架而非知识点的堆砌

### 5. 《商业洞察力》（2020）

**核心论点**：
- 洞察力 = 看透事物本质的能力
- 好的洞察力来自系统思考 + 第一性原理
- "你看到的是现象，洞察看到的是本质"
- 用系统动力学分析商业问题：变量、因果链、增强回路、调节回路、滞后效应

**关键框架（系统动力学5要素）**：
1. 变量：系统中的关键要素
2. 因果链：变量之间的关系
3. 增强回路：正反馈循环（飞轮效应）
4. 调节回路：负反馈循环（自我约束）
5. 滞后效应：原因和结果之间的时间差

### 6. 《新零售》（2018）

**核心论点**：
- 新零售的本质 = 效率更高的零售
- 用"信息流、资金流、物流"三流重构理解零售变革
- 线上线下不是对立关系，而是效率的不同实现方式

## 二、跨著作反复出现的核心论点

以下论点在多部著作中反复出现，具备"心智模型"候选资格：

### 2.1 系统思考
- 出现在：《商业洞察力》（核心主题）、《底层逻辑》、《5分钟商学院》
- 一致性：高度一致，从未改变立场
- 核心主张：看系统而非看要素，看连接而非看个体

### 2.2 进化思维
- 出现在：《进化的力量》（核心主题）、年度演讲系列
- 一致性：持续深化，从商业策略扩展到人生哲学
- 核心主张：适应环境的能力比当前状态更重要

### 2.3 效率驱动
- 出现在：《新零售》、《5分钟商学院》、几乎所有商业分析
- 一致性：核心锚点，从未动摇
- 核心主张：商业的终极竞争是效率的竞争

### 2.4 概率思维
- 出现在：《底层逻辑》、《5分钟商学院》
- 核心主张：不做确定性判断，做概率判断；选择大概率事件

### 2.5 第一性原理
- 出现在：《底层逻辑》、《商业洞察力》
- 核心主张：回到最基本的事实和公理，从零推导，不接受"因为大家都这样做"

### 2.6 复利思维
- 出现在：几乎所有著作
- 核心主张：长期坚持做正确的事，时间会放大效果

### 2.7 价值公式
- 出现在：《底层逻辑》核心章节
- 公式：价值 = 功能价值 + 情绪价值 + 资产价值
- 商业分析中反复使用此三维度拆解
