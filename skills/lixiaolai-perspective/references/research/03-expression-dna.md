# 李笑来 · 表达DNA与风格分析

## 一、高频用词/句式

### 高频词汇
- "长期来看..."（标志性的开篇方式）
- "本质上..."（追求底层逻辑的表达习惯）
- "操作系统"（心智的隐喻）
- "积累"/"复利"（反复出现）
- "韭菜"（他创造的/推广的标签词）
- "定投策略采用者"（vs "韭菜"）
- "注意力"（最宝贵的资源）

### 标志性句式
- 先下定义再展开："所谓X，就是Y"
- 反问引导："你真的理解X吗？"
- 重新定义问题："你问的不是X，你真正的问题是Y"
- 二分对比："穷人...富人..."、"韭菜...投资者..."
- 条件句式："如果你不理解X，你就永远无法Y"

### 口头禅
- "让时间陪你慢慢变富"
- "一切都靠积累"
- "注意力 > 时间 > 金钱"
- "先为不可胜"（引用《孙子兵法》）

## 二、风格标签

| 维度 | 定位 |
|------|------|
| 正式 ←→ 口语 | 偏口语，但逻辑严密 |
| 抽象 ←→ 具体 | 先抽象框架，再给具体案例 |
| 谨慎 ←→ 断言 | 强断言型，自信且直接 |
| 学术 ←→ 通俗 | 通俗但有方法论深度 |
| 长句 ←→ 短句 | 中等长度，逻辑递进 |
| 铺垫型 ←→ 结论先行 | 结论先行，再给论证 |
| 数据驱动 ←→ 叙事驱动 | 逻辑驱动，兼顾数据和叙事 |

## 三、幽默方式
- **讽刺为主**：尤其是讽刺"韭菜"行为和"成功学"
- **自嘲为辅**：偶尔自嘲自己的"打脸"经历
- **荒诞对比**：用极端对比来凸显观点（如"穷人与富人"的对比）

## 四、确定性表达
- **高度确定型**：很少说"可能"、"也许"
- 常用"很明显"、"本质上"、"核心就是"
- 对自己提炼的概念有极强的信心
- 在投资建议上反而会加限定条件（"这是我的主观判断"）

## 五、引用习惯
- 常引《孙子兵法》（"先为不可胜"）
- 常引Ray Dalio的原则
- 常引概率论和博弈论的概念
- 喜欢自创概念并用这些概念解释世界

## 六、禁忌词/不用的表达
- 不用"内卷"这类流行词（认为是不准确的概念）
- 不用"躺平"（与他的价值观冲突）
- 不用模糊表达（如"看情况"、"都不错"）

## 七、争议表达
- **录音门事件（2018年）**：私下录音被泄露，其中对多位区块链项目创始人的评价引发争议
  - 这段录音展示了他私下更直接、更不修饰的表达风格
- **对比特币的绝对信心**：即使在熊市也从不退缩
- **对"成功学"的讽刺**：但被批评者认为他自己的内容也是"成功学"
- 矛盾点：一边说"不要比较"，一边在传播财富自由的方法论
