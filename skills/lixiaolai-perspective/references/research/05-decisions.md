# 李笑来 · 决策记录与行动

## 一、重大投资决策

### 1. 首次接触并投资比特币（约2011年）
- **背景**：比特币价格极低时期
- **决策逻辑**：看到了比特币作为"数字黄金"的长期价值，认为这是人类历史上第一次用技术手段实现了私有财产的不可侵犯
- **结果**：大量低价买入，后续成为"中国比特币首富"
- **事后反思**：这一决策奠定了他投资哲学的基础——长期持有、不为短期波动所动
- **来源可信度**：混合（本人提及+媒体报道）

### 2. 创建BOX定投组合（2019年）
- **背景**：在《定投改变命运》中提出
- **决策逻辑**：选择BTC、EOS、XIN作为组合，掺杂了他的主观判断
- **结果**：EOS表现不及预期，BTC大幅上涨
- **言行一致性**：他鼓励读者根据自己的认知选择投资组合，不强制跟投BOX
- **来源可信度**：一手（GitHub开源书籍）

### 3. 长期持有策略
- **决策逻辑**：不做短线投机，"短期价格无关紧要"
- **实际行为**：据报道持有约10万枚BTC的远古钱包动向仍受市场关注
- **一致性**：高度一致——他的行为与公开主张的"长期持有"理念吻合
- **来源可信度**：二手（媒体报道）

### 4. 2018年录音门后的低调
- **背景**：录音泄露引发巨大争议
- **决策**：逐渐淡出公众视野，进入"闭关"状态
- **事后行为**：将重心从区块链投资转向家庭教育
- **来源可信度**：二手（多方报道）
