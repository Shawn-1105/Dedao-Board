# 李笑来 · 完整时间线

## 关键节点

| 时间 | 事件 | 对思维的影响 |
|------|------|-------------|

| 约1990年代 | 延边大学就读 | 形成基础思维方式 |

| 2009年 | 出版《把时间当作朋友》 | 提出"心智"概念和"积累+复利"框架；确立"反成功学"立场 |

| 2013年前后 | 离开新东方，成为独立作者/投资者 | 从"卖时间"到"卖系统"的转型——验证《财富自由之路》中"不卖时间"的理念 |
| 2016年 | 在得到App开设"通往财富自由之路"专栏 | 系统化输出投资和思维方法论；提出"注意力>时间>金钱"价值排序 |
| 2017年 | 区块链热潮中成为焦点人物 | 信息不对称的实践——在行业最热时保持冷静判断 |
| 2018年 | 录音门事件爆发 | 争议促使反思；开始从"投资布道者"转向更深层的思考 |
| 2018年 | 出版《韭菜的自我修养》 | 将投资认知系统化；提出"反韭菜"思维框架 |
| 2019年 | 开源《定投改变命运》（GitHub） | 提出BOX定投组合；将"定投是普通人唯一正确的投资方式"系统化 |
| 约2020年 | 开始转向家庭教育领域 | 将投资思维（长期主义、复利、产权）融入家庭教育方法论 |
| 约2021年 | 出版/开设《教练的真相》课程 | 重新定义父母角色为"教练"；提出"教是最好的学" |
| 约2022-2023年 | 推出"好的家庭教育"课程体系 | 六个基础观念形成完整体系：尊重产权、崇尚生产、直面困境、决策担当、痴迷改进、健康第一 |
| 约2023年 | 出版《好的家庭教育》 | 系统化家庭教育理念 |
| 2024年 | 比特币现货ETF在美国获批 | 验证了他对比特币长期价值的一贯判断 |
| 2025年7月 | "重出江湖"B站访谈 | 分享闭关期间最新认知；AI时代的新思考；家庭教育实践体会 |

## 思想演变轨迹

### 投资思想
- 2011：发现比特币价值（直觉判断）
- 2011-2018：实践长期持有（行为验证）
- 2018：《韭菜的自我修养》（认知系统化）
- 2019：《定投改变命运》（方法论输出）
- 2024-2025：BTC ETF获批验证判断（信念被市场验证）

### 教育思想
- 2001-2013：新东方教学实践（教是最好的学）
- 2009：《把时间当作朋友》（心智管理方法论）
- 2016：得到专栏（成人教育/思维升级）
- 2020-2023：转向家庭教育（将投资思维融入教育）
- 2025：继续深化家庭教育实践

### 核心不变的思想
- 长期主义（从2009年至今从未改变）
- 积累+复利（贯穿所有领域）
- 自学能力的重要性（从教学到投资到教育）
- 注意力是最宝贵的资源

## 最新动态（2025年）
- 7月B站"重出江湖"访谈
- 持续运营家庭教育课程体系
- 对AI时代的思考（独立思考能力更加重要）
- 投资方面：长期持有策略未变，BTC ETF获批验证了早期判断
