# 查理·芒格 · 时间线

| 时间 | 事件 | 对思维的影响 |
|------|------|-------------|
| 1924年1月1日 | 出生于内布拉斯加州奥马哈 | 起点——与巴菲特同乡 |
| ~1940年代 | 密歇根大学学习数学/物理 | 数学思维、物理思维的基础 |
| 1943年 | 美国陆军航空队服役 | 二战经历——对风险和不确定性的早期理解 |
| ~1946年 | 在加州理工学院学习气象学 | 科学方法论、概率思维 |
| 1948年 | 哈佛法学院毕业（无本科学位） | 法律思维的训练——系统论证、逆向分析 |
| ~1960年代 | 在洛杉矶从事律师工作 | 法律分析能力；接触大量企业案例 |
| 1962年 | 与巴菲特首次深度交流 | 开始合作——巴菲特称芒格扩展了他的视野 |
| ~1962-1975年 | 经营惠勒·芒格投资合伙公司 | 投资实践；验证"以合理价格买优秀企业"的理念 |
| 1978年 | 成为伯克希尔·哈撒韦副董事长 | 与巴菲特正式合作，开始传奇投资旅程 |
| ~1980年代 | 推动巴菲特从"烟蒂投资"转向"优质企业" | 投资哲学的根本转变——伯克希尔最关键的转折 |
| 1986年 | 在哈佛学校毕业典礼演讲"如何保证人生失败" | 逆向思维方法论的标志性阐述 |
| 1988年 | 推动投资可口可乐 | 品牌护城河理论的经典实践 |
| ~1990年代 | 持续完善多元思维模型体系 | 80-90个核心思维模型成型 |
| 1994年 | USC商学院演讲"世俗智慧" | 系统阐述多元思维模型方法论 |
| ~2000年代 | 系统阐述"人类误判心理学" | 25个认知偏误的完整框架 |
| 2005年 | 《穷查理宝典》出版 | 思想体系被系统记录 |
| 2008年 | 投资比亚迪 | 跨越传统价值投资边界——承认这是一次"风投" |
| 2015-2023年 | Daily Journal年度会议演讲 | 持续输出投资和人生智慧；对加密货币强烈反对 |
| 2023年11月28日 | 去世，享年99岁 | 遗产：一生的智慧结晶 |

## 思想演变轨迹

### 早期（1940s-1960s）
- 法律训练→系统分析能力
- 数学/物理训练→概率思维

### 中期（1960s-1990s）
- 与巴菲特合作→投资哲学成型
- 从"烟蒂投资"→"优质企业投资"
- 多元思维模型体系建立

### 成熟期（1990s-2023）
- 人类误判心理学完善
- 人生智慧的系统输出
- 跨学科思维成为本能

## 核心不变的思想
- 逆向思维（Inversion）—— 从1940年代法律训练到去世
- 终身学习—— 从青年到99岁从未停止阅读
- 诚实守信—— 贯穿整个人生
- 跨学科思维—— 从法律到投资到心理学
- 耐心与纪律—— 投资和人生的统一原则
